## вЪЩпЄП Installation

### Clone the Repository

```bash
git clone https://github.com/gadadasu-jahnavi/popx-react
```

### Navigate to Project Folder

```bash
cd popx-react
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```
